# Module 1 Budget Tracker (Rust)

## Overview
This is a simple Rust program built to practice **Module 1 concepts**:
- Variables and mutability
- Constants
- Arrays
- Loops (`for`)
- Conditional statements (`if`, `else`)
- Expressions and statements
- Shadowing

The program simulates **daily expenses against a wallet balance**, enforcing a **daily spending limit**. It tracks **successful and failed transactions** and prints a **final summary**.

---

## Features
- Hardcoded expenses array
- Daily limit check
- Balance update after each transaction
- Success and failure counters
- Day tracking for clarity
